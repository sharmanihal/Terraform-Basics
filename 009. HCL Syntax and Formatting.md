#### **1. What is HCL?**

* **HCL (HashiCorp Configuration Language)** is the language used by **Terraform** to describe and define infrastructure.
* It represents **infrastructure objects** like providers, resources, variables, etc., in a **human-readable** format.

---

#### **2. Basic Block Structure**

In HCL, everything is written in **blocks**.

**General syntax:**

```hcl
<block_type> <label_1> <label_2> {
  <identifier> = <expression>
}
```
#### **Example**

```hcl
resource "aws_instance" "app_server" {
  ami           = "ami-830c94e3"
  instance_type = "t2.micro"
}
```
